How to run this project:

1. Navigate to the folder "Portfolio/app/"
2. Open a terminal.
3. Run the command "python main.py"
4. If you are on your local machine, visit: http://0.0.0.0:5000
   If you wish to connect from another machine: http://<server ip address>:5000
     where <server ip address> is the ip address of the server.
